Run Locally

From the repository root:

pip install -r requirements.txt

python web_app.py

Then open http://127.0.0.1:8050.

Hardware Setup

  • Connect PlutoSDR TX to RX with SMA cable.
  • Use a 30 dB attenuator before RX.
  • In the web dashboard, choose Hardware (PlutoSDR) and click Connect.
